## 📌 Project Overview

This is a console-based **Rock-Paper-Scissors Game** developed using **Python** as part of my **Python Programming Internship at CodSoft**.

The game allows the user to play against the computer with real-time score tracking, input validation, and an interactive command-line interface. It demonstrates the use of core Python programming concepts such as functions, loops, conditional statements, and the built-in `random` module.

## 🎯 How the Game Works

1. Enter one of the following:
   - `rock`
   - `paper`
   - `scissors`
   - `exit`

2. The computer randomly selects one option.

3. The game compares both choices and determines the winner.

4. Scores are updated after each round.

5. The game continues until the user types **exit**.
